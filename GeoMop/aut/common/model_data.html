model_data.autoconversion module¶

GeoMop model auto-conversion module

Ensures auto-conversion of data for specified format.

model_data.autoconversion.autoconvert(node, input_type)¶

Performs recursive auto-conversion on root node.

Auto-conversions:
  1. If Array is expected and scalar/record is found, either perform a transposition or encapsulate it in Array(s).
  2. If Record is expected and scalar/array is found, check reducible_to_key. If present, create the Record.
  3. If Abstract is expected and scalar/array is found, check if default_descendant fits rule 2.

The function also converts data to the expected data types (if possible).
